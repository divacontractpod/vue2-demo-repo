# Vue 2 + Weblate Demo

## Setup
```
npm install
npm run serve
```

## How it works
- Vue 2 app uses vue-i18n
- Translations are stored in src/locales/*.json
- Weblate manages these JSON files via Git

## Connect to Weblate
- File mask: src/locales/*.json
- Source file: src/locales/en.json
- Format: JSON

## Switch languages
Use buttons in UI to toggle between EN and FR.
